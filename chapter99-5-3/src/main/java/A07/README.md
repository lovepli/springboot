#  多态

    最终多态体现为父类引用变量可以指向子类对象：父类类型 变量名 = new 子类类型(); 
    多态的条件：
        1、多态的前提是必须有子父类关系或者类实现接口关系，否则无法完成多态。 
        2、在使用多态后的父类引用变量调用方法时，会调用子类重写后的方法。 
     
     
     五、多态的好处与弊端 
     当父类的引用指向子类对象时，就发生了向上转型，即把子类类型对象转成了父类类型。
     向上转型的好处是隐藏了子类类型，提高了代码的扩展性。
